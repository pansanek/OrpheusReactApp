# ORPHEUS — Умная Мобильная Платформа Социальной сети для Музыкантов

## Краткое описание

**ORPHEUS** — это специализированная социальная сеть для музыкального сообщества, созданная для решения ключевых проблем музыкантов на всех этапах карьеры. Платформа объединяет музыкантов, музыкальные учреждения и мероприятия в единой цифровой экосистеме.

**Основная цель**: автоматизировать поиск партнёров для музыкальных проектов, сократить время на формирование коллективов на 40-60% и создать безопасную среду для коммуникации.

## Быстрый старт

### Установка и запуск

```bash
# Установка зависимостей
npm install
# или
yarn install

# Запуск в режиме разработки
npm start
# или
yarn start

# Сборка для production
npm run build
# или
yarn build
```

Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)

##  Ключевые функции

### Управление профилями
- **Расширенный музыкальный профиль** с инструментами, жанрами, уровнем навыков (1-5)
- **Цифровое портфолио** для демонстрации творчества
- **Система тегов для ИИ-рекомендаций** (пользовательские интересы)

### Поиск и рекомендации
- **Умный поиск** музыкантов по критериям: инструмент, жанр, уровень, местоположение
- **Персонализированные рекомендации** на основе:
  - Совпадения инструментов/жанров
  - Совпадения пользовательских тегов ИИ
  - Геолокации
- **История поиска** для улучшения рекомендаций

### Управление группами
- **Создание музыкальных коллективов** с детальным описанием
- **Управление членством** и ролями (лидер, участник)
- **Совместный календарь** репетиций и мероприятий
- **Групповая лента** постов и обсуждений

### Социальные взаимодействия
- **Лента постов** с поддержкой текста, фото, видео, аудио
- **Система реакций** (лайки, комментарии)
- **Личные и групповые чаты**
- **Автоматическая модерация** контента

### Каталог учреждений
- **База репетиционных студий**, концертных площадок
- **Система бронирования** с календарём доступности
- **Отзывы и рейтинги**
- **Фильтры** по оборудованию, цене, местоположению

### Теги для ИИ-рекомендаций
- **Управление персональными тегами** ("ищу джем-партнёров", "готов к гастролям")
- **Категоризация тегов**: активность, коллаборация, локация, навык, цель
- **Визуализация** цветными бейджами
- **Влияние на рекомендации** музыкантов и мероприятий


## Адаптивность

Приложение полностью адаптивно и оптимизировано для:
- **Мобильные устройства** (320px - 767px)
- **Планшеты** (768px - 1023px)
- **Десктопы** (1024px+)

## Тестирование

### Ключевые пользовательские сценарии
1. **Регистрация → редактирование профиля → добавление тегов ИИ**
2. **Поиск музыкантов → просмотр профиля → приглашение в группу**
3. **Создание группы → публикация поста → взаимодействие в ленте**
4. **Поиск студии → "бронирование" → уведомление**

### Запуск тестов
```bash
npm test
# или
yarn test
```
